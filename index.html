Connect Google Drive

PackFit needs a Google OAuth Client ID to sign you in and sync your wardrobe & trips to Drive. This is free and takes ~5 minutes.

  1. Go to console.cloud.google.com
  2. Create a new project (or select one)
  3. Go to APIs & Services → Library → enable Google Drive API
  4. Go to APIs & Services → OAuth consent screen → set to External, fill in app name
  5. Go to Credentials → Create Credentials → OAuth client ID
  6. Choose Web application, add your site URL to Authorised JavaScript origins
  7. Copy the Client ID (ends in .apps.googleusercontent.com) and paste below

Your Client ID is stored only in your browser's localStorage. It never leaves your device.
